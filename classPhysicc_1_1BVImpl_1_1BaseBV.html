Detailed Description

template<typename Derived, typename BoundingObject>
class Physicc::BVImpl::BaseBV< Derived, BoundingObject >

A templated class that defines the Bounding Volume (BV) of an object, but in a way that allows hot swapping actual bounding volumes (like AABBs, OBBs, 8-DOPs, etc.).
